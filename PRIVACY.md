# Privacy Policy for Snipply

Snipply is a Chrome extension that lets users create text snippets and replace typed shortcuts with saved text.

## Data stored by the extension

Snipply stores the following data locally in the user's browser using `chrome.storage.local`:

- Snippet names
- Snippet shortcuts
- Snippet text
- Folder names
- Folder descriptions
- Folder icons
- Extension language settings

This data is stored locally on the user's device through Chrome extension storage.

## Website content

Snipply runs on web pages so it can detect shortcuts typed by the user inside editable fields and replace them with the corresponding saved snippet.

Snipply only uses editable field content for this local shortcut replacement feature.

## Data sharing

Snipply does not sell, transfer, or share user data with third parties.

Snipply does not send snippets, website content, browsing history, or settings to any external server.

## Remote code

Snipply does not use remote code. All code required by the extension is included in the extension package.

## Contact

For questions about this privacy policy, contact the developer through the Chrome Web Store listing or the project repository.
